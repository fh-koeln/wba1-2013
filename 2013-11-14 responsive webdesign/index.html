Media Queries

  • Media Queries = Medienabfragen
  • in CSS3 implementiert
  • verschiedene Medientypen können zugeordnet werden:
  • z.B. all (default), print oder screen
@media screen {
/* INHALT */
}
  • Medienmerkmale beschreiben die Anforderungen an den Medientyp
  • width
    • @media screen and (width: 600px){
/* INHALT */
}
  • height
    • @media screen and (height: 600px){
/* INHALT */
}
  • device-width / device-height
    • @media screen and (device-width: 800px){
/* INHALT */
}
  • jeweils eigenständig oder mit Präfix min- und max-
    • @media screen and (max-width: 500px){
/* INHALT */
}
  • orientation - kann landscape oder portrait
    • @media screen and (orientation: landscape){
/* INHALT */
}
                        
@media (max-width:500px) {
/* INHALT */
}
  • Wieviel Platz haben wir im Browserfenster?
  • Breakpoint: max-width als Pixel
  • horizontaler Platz ist am wichtigsten
  • Inhalt innerhalb der geschweiften Klammern 
@media (max-width:500px) {    
    .wrapper {
        width:100%; 
    }
    ul#container{
        position:relative; 
        height:100%;
        background-color:transparent;
    }
}
