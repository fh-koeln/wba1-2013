Media Queries

  • Media Queries = Medienabfragen
  • in CSS impelemtiert
@media (max-width:500px) {
/* INHALT */
}
  • Breakpoint: max-width
  • als Pixel
  • Inhalt innerhalb der geschweiften Klammern 
@media (max-width:500px) {    
    .wrapper {
        width:100%; 
    }
    ul#container{
        position:relative; 
        top:10px;
        left:0px;
        height:100%;
        background-color:transparent;
        list-style:none;
    }
}

Responsive Images

  • Bilder sollten sich anpassen
  • Größen (Höhe, Breite) in Prozent angeben
  • in CSS impelemtiert
