Willkommen zu JavaScript

JavaScript (kurz JS) ist eine Skriptsprache, die...

  • ursprünglich für dynamisches HTML in Web-Browsern entwickelt wurde
  • Benutzerinteraktionen auswertet
  • Inhalte verändert, nachlädt oder generiert (Clientseitig)
  • die Möglichkeiten von HTML und CSS erweitert

DOM (Document Object Model)


Was ist DOM eigentlich?


  1. Schnittstelle für den Zugriff auf HTML- oder XML- Dokumente
  2. DOM ist keine eigene Programmiersprache
  3. DOM ist nicht auf HTML beschränkt
  4. Definiert Objekte, Eigenschaften und Methoden, die eine Programmiersprache umsetzen soll
  5. Dom wird von allen gängigen Browsern verwendet

Funktionen in JavaScript

  • sind wichtige Bausteine von Javascript
  • ein Block von Anweisungen mit einem Namen
  • Der Funktionsblock wird einmal definiert
  • Das Zusammenfassen von Befehlen in einer Funktion verbessert die Lesbarkeit des Scripts
							
function groesseVeraendern(obj){
	document.getElementById(obj).style.width = "250px";
	document.getElementById(obj).style.height = "200px";
}
  • Schlüsselwort: function
  • Name: groesseVeraendern
  • Parameter: (obj)
  • Funktionskörper: { Anweisungen }
