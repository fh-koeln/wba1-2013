    Aufbau HMTL

  • Ein HTML Element beginnt mit einem Eröffnungs-tag <p>
  • Ein HTML Element endet mit einem Abschuss-tag </p>
  • Der Elementinhalt ist alles zwischen dem Eröffnungs-/ und Abschluss-tags
    <p>Elementinhalt</p>
  • Einige Elemente enthalten leeren Inhalt und besitzen keinen Abschluss-tag<img>

    Wie baut sich eine HTML Seite auf?


    Ein HTML Dokument besteht im großen und ganzen aus fünf verschiedenen Einzelteilen :


    <!DOCTYPE html>
    <html></html>
    <head></head>
    <title></title>
    <body></body>

    Aus diesen Basiselementen ergibt sich das Grundgerüst einer HTML-Seite.

    Ein DOCTYPE DEAL ?

    Die Angabe zur verwendeten HTML-Version


    Alter DOCTYPE (Version 4.01):

    !DOCTYPE HTML PUBLIC -//W3C//DTD HTML 4.01 Transitional//EN http://www.w3.org/TR/html4/loose.dtd>



    Neuer DOCTYPE (ab Version 5.0):

    !DOCTYPE html



    DEAL!

    Das HTML-Tag


    <html></html>

    • Teilt dem Browser mit, das es sich um ein HTML-Dokument handelt.
    • Umgibt alle anderen Tags / Wurzelelement.
    • Beginnt und beendet das HTML Dokument.

    Das HEAD-Tag


    <head></head>

    • Bietet Platz für sieben weitere HTML-Elemente mit verschiednene Aufgaben.


    title Titel der Seite
    meta diverse Metainformationen
    base Angabe der Basis-URL
    link Einbindungen von anderen Resourcen, z.B. Stylesheets
    script Einbindung von "fremdem" Code, z.B. JavaScript
    style Stilinformationen / CSS-Deklarationen
    object Einbinden von externen Dateien

    Das HEAD-Tag

    Das TITLE-Tag


    <title></title>

    • Befindet sich innerhalb des HEAD-Bereichs der HTML-Datei.
    • Enthält den Pagenamen.
    • Anzeige erfolgt in einem Browser-Tab oder Titelleiste.

    Das BODY-Tag


    <body></body>

    • Sichtbarer / dargestellter Teil eines Dokumentes im Browser.
    • Eigentlicher Inhalt des HTML-Dokumentes.

    Block & Inline Elemente

    Block Elemente

    • Erzeugen immer einen neuen Absatz
    • Können weitere HTML Elemente beinhalten
    • Beipiel: Headings, Paragraph, Division (div)

    Inline Elemente

    • Erzeugen keine neue Absätze
    • Können nur andere Inline Elemente beinhalten
    • Beispiel: Strong, Span, Break

    Div Block

    <div> element </div>
    • Behälter (container)
    • Struktur & Manipulation

    Beispiel: 

     
<div>
		
	<p> Paragraph 1 </p>
	<p> Paragraph 2 </p>
					
</div>
					
     
<div align="right">
		
	<p> Paragraph 1 </p>
	<p> Paragraph 2 </p>
					
</div>

    Ausgabe:

    Paragraph 1

    Paragraph 2

    Images & Anchor points

    Images

     <img src="wo das Bild ist">
    • Verknüpfung von Bilder
    • Source Attribut immer nötig: src

    Beispiel: 

     
<img src="images/Cookie.gif">

    Anchor points

     <a href="link"> </a>
    • Verlinkung zu Webseiten, Dokumente, Teile der aktuellen Seite, E-mails
    • Referenz Attribut immer nötig: href

    Beispiel: 

     
<a href="http://www.seite.de"> Meine Seite </a>

    Meine Seite

    Oder E-mailadresse mit Attribut: mailto 

    
<a href="mailto:meineAdresse@irgendwo.de"> Meine E-mail </a>

    Meine E-mail
