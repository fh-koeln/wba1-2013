2. Positionierung

4 Arten der Positionierung

  • position: static;
  • position: relative;
  • position: absolute;
  • position: fixed;

2.1 position:static - Hier bleibt alles beim alten

  • Default für Attribut position
  • Statisch, d.h. nicht mittels Versatzeigenschaften positionierbar
  • Inhalte erscheinen in Reihenfolge,
    in der sie im HTML-Quelltext vorliegen.


selektor {
    position: static;
    bottom: 20px; /*Das würde nichts bewirken*/
    left: 15%;    /*Auch überflüssig*/
}

2.2 position:relative - Versteckte Kräfte

  • Verhält sich wie static
  • Versatzeigenschaften top,right,bottom,left verfügbar
  • Koordinatensystem orientiert sich an statischer Position
  • Versatz beinflusst Position umliegender Elemente nicht


selektor {
    position: relative;
    bottom: 20px;
    left: 15%;
}

2.3 position:absolute - Überall, jederzeit

  • wird aus normal flow entfernt, d.h. die Position:
    • wird nicht durch andere Elemente beeinflusst
    • beeinflusst andere Elemente nicht
  • Koordinatensystem orientiert sich am nächsthöheren Elternelement =/ position:static im Markup
  • Versatzeigenschaften top,right,bottom,left verfügbar
    • selektor {
    position: absolute;
    bottom: 20px;
    left: 15%;
}


2.4 position:fixed - Immer genau da

  • Fast alles wie bei absoluter Positionierung
  • Koordinatensystem jedoch immer gemessen am "Viewport" (Browserfenster, Bildschirm mobiler Geräte)
    • d.h es bewegt sich nicht mit, wenn gescrollt wird


selektor {
    position: fixed;
    bottom: 20px;
    left: 15%;
}

2.5 Weitere nützliche Positionierungs-Attribute

  • float & clear
  • Bsp.:
selektor {
    float:left;

2.6 Der z-index

Was wenn Elemente übereinander liegen?

  • Zuletz genanntes Element im HTML Code liegt vorne
  • konfigurierbar über Bsp.:
rot {
    z-index: 2;
}

blau {
    z-index: 3;
}

gruen {
    z-index: 4;
}
  • Element mit höherem z-index überdeckt Elemente mit niedrigerem z-index

  • ::first-line

  • ::first-letter
    erstes Zeichen des Textes erhält CSS-Eigenschaften
  • ::before
    fügt Inhalt vor Element ein
  • ::after
    fügt Inhalt nach Element ein

4. Kombinatoren

  • Elemente stehen zueinander in verwandtschaftlicher Beziehung
    • Bedingungsselektor Zielselektor {
    ...
}
  • Bedingung erfüllt wenn Verwandtschaftsbeziehung zum Zielselektor zutreffend

4.1 Arten von Kombinatoren

  • Nachfahren-Kombinator
    • wählt alle direkten & indirekten Nachfahren des Bedingungsselektors aus, die dem Zielselektor entsprechen
      • Bedingungsselektor Zielselektor {
    ...
}
  • Kind-Kombinator
    • wählt alle direkten Nachfahren des Bedingungsselektors aus, die dem Zielselektor entsprechen
      • Bedingungsselektor > Zielselektor {
    ...
}

4.2 Arten von Kombinatoren

  • Nachbar-Kombinator
    • wählt alle direkten Nachbaren des Bedingungsselektors aus, die dem Zielselektor entsprechen
      • Bedingungsselektor + Zielselektor {
    ...
}
  • Geschwister-Kombinator
    • wählt alle direkten & indirekten folgenden Nachbaren des Bedingungsselektors aus, die dem Zielselektor entsprechen
      • Bedingungsselektor ~ Zielselektor {
    ...
}

5. Attribut-Selektoren


"Immer wieder soll ein HTML-Element mit CSS-Stilen gezielt ausgewählt und geändert werden. Wenn es eine CSS-Klasse oder ein ID-Attribut gibt, ist das Ansteuern eines HTML-Tags mit CSS einfach."

Aber wie steht es mit einem Element mitten in einer Liste?

a.blue {
    ...
}

entspricht

a[class~="blue"] {
    ...
}

5.1 Attribut-Selektoren

a[title]
 Passt auf Element a mit Attribut title. 
a[title="red"]
 Passt auf Element a mit Attribut title und exakten Wert red. 
a[title~="red"]
 Passt auf Element a mit Attribut title, dessen durch Leerzeichen getrennte Liste von Werten red enthält. Auch auf Attribut-Wert "red little flexible" zutreffend. 
a[title|="red"]
 Passt auf Element a mit Attribut title, dessen durch Trennstriche (-) getrennte Liste von Werten red enthält. 
a[title^="red"]
 Passt auf Element a mit Attribut title, dessen Wert mit red beginnt. Auch auf Attribut-Wert reddish zutreffend. 
a[title$="red"]
 Passt auf Element a mit Attribut title, dessen Wert mit red endet. Auch auf Attribut-Wert rosered zutreffend. 
a[title*="red"]
 Passt auf Element a mit Attribut title, dessen Wert red an beliebiger Stelle in Zeichenkette enthält

7. Transitions

  • ermöglichen es, Änderung eines Elements von einem Style zum anderem zu steuern
  • Solch eine Änderung geschieht beispielsweise beim :hover über ein Element.
div {
    transition-property: width; /*Property, auf das transition anzuwenden ist (all=alle Properties)*/
    transition-duration: 1s; /*Dauer der Transition-Animation */
    transition-timing-function: linear; /*Timing-Funktion gibt zeitl. Verlauf der Animation vor*/
    transition-delay: 2s; /*Delay bevor Transition einzusetzen hat*/
}
