1. Was ist HTML?

  • Hyper
  • Text
  • Markup

  • Language


  • keine Programmiersprache
  • Beschreibung von Strukturen, Inhalte und das Verhalten eines Dokuments
  • optische Umsetzung durch Browser
  • Aktuell: HTML5

2. Historie

  • Erfindung in CERN von Tim Beners-Lee (1990)
  • HTML 1.0 Darstellung von Grafiken, Überschriften, Dokumente und Verweise (z.B. Links)
  • HTML 2 Sonderzeichen, sonst keine großartigen Neuerungen (1995)
  • HTML 3.2 Tabellen und Container (1997)
  • HTML 4 & 4.01 Optische Gestaltung auf CSS verlegt (1998)
  • HTML5 Release (2014)
