Einbindung von jQuery


<head>
	<script type="text/javascript" src="jquery-2.0.3.js"></script>
</head>

jQuery wird im Head eingebunden

  • blockiert die Seite
    • Lösung: nach dem Content

Document Ready

Seite musst vollständig geladen sein oder die DOM Hierarchie musst komplett aufgebaut sein, damit der JavaScript Code ausgeführt wird.


Funktion ready()

$(function() {
	// Weitere Eventhandler können hier rein.
});

  • garantiert DOM vollständig
  • weitere Eventhandler/jQuery Code einfügen

Beispiel

<!doctype html>
<html>
<head>
  <meta charset="utf-8">
  <title>ready() Beispiel</title>

  <script src="jquery-2.0.3.js"></script>
  <script>
  $(function() {
    $( "p" ).text( "Die DOM ist jetzt geladen und kann nun manipuliert werden." );
    //Ersetzt den Text im Body, wenn die Seite fertig geladen ist.
  });
  </script>
</head>
<body>
 
	<p>Noch nicht geladen.</p>

</body>
</html>

Syntax


$(selector).action()

$ := jQuery-Notation
selector := CSS-Selector
action() := jQuery-Funktion


Beispiele:
$(this).hide() - Blendet das vorliegende Element aus.

$("p").hide() - Blendet alle <p> Elemente aus.

$("#test,p").hide() - Blendet Elemente mit der id "test" und alle <p> Elemente aus.

$(".test").hide() - Blendet alle Elemente mit der class="test" aus.
