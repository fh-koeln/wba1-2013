Was ist CSS?


    Cascading Style Sheets

    = übergeordnete Stilvorlagen



    - Formatierung von HTML-Elementen


    - Trennung von Struktur und Gestaltung:

    -> HTML sorgt für Struktur
    -> CSS sorgt für Gestaltung
    - aktuell: CSS 3
    seit 2000 in Entwicklung

Chancen


    - mehr Möglichkeiten der Gestaltung
    Bsp.: Schriftgestaltung, Rahmen, Innen- und Außenabstände, ...


    - bessere Aktualisierbarkeit

    - Accessibility / Barrierefreiheit

    - Corporate Design

Risiken


    - Unterschiedliche Unterstützung durch Browser

Vererbung


    HTML-Dokument gleicht einer Baumstruktur:

    Vererbungsprinzip gilt auch für CSS:
    body
{
  font-family: Arial;
  color: #fff;
  text-align: center;
}

Aufbau

  • Befehl

  • Deklaration

Was ist ein Befehl?


    - In CSS wird mit Befehlen gesagt, was in einem speziellen Bereich getan wird.


    CSS-Befehl Bespiele:

    Schriftart Arial:
    font-family: arial;

    Textausrichtung Linksbündig:
    text-align: left;

    Hintergrundfarbe Schwarz:
    background-color: #000;


Was ist eine Deklaration?


    - Die Deklaration bestimmt, wie etwas formatiert werden soll.
    Sie besteht immer aus einer Eigenschaft und einem Wert.


    Beispiel:
    
        h1{
            color:#fff;
            }

    Die Deklaration sagt, dass die Syntax mit einem Selektor (h1) anfängt.
    -> Schriftgröße h1

     In der geschweiften Klammer steht der Befehl in Form von einer Eigenschaft :
    -> Schrift mit Größe h1 hat Farbe #fff

Was ist eine Eigenschaft?


    - Die Eigenschaft sagt uns, wo die Veränderung stattfinden soll.


    Beispiel:
    
background-color:#000;

Was ist ein Wert?


    - Wert wird Eigenschaft zugeordnet.


    Beispiel:
    
background-color:#000;

Was sind Einfache Selektoren


    - Tag-Namen wie z.B. p, div oder body kann man damit direkt ansprechen.

     p { color: red; }

    -> sorgt dafür, dass alle p-Tags eine rote Schriftfarbe haben.

Klassen-Selektoren


    Möchte man jedoch nicht alle Tags ansprechen, kann man diesen Klassen zuweisen.

    Text

    Diese Klasse ruft man dann mit einem Punkt in der CSS auf

     .inhalt{color: red;}

    Wenn man aber mehrere tags mit dem selben Klassen-Namen hat

     p.inhalt{color: red;}

    Man kann auch mehrere Klassen definieren und gleichzeitig verwenden

     
    Text

ID-Selektoren


    Das ganze Funktioniert auch mit IDs. 

     
    Text2

    Diese werden durch Rauten im CSS angesprochen 

     #inhalt{color: red;}

    ID´s müssen eindeutig sein => dürfen nur jeweils einmal im Dokument vorhanden sein

Das Einbinden von CSS-Dateien


  • internes Einbinden

  • externes Einbinden

internes Einbinden


    Das interne Einbinden von CSS-Dateien in vorhandene HTML-Elemente kann zum einen in der Header-Datei der jeweiligen Website, zum anderen im Attribut selbst erfolgen. 
    
<!DOCTYPE html>
<html>
	<head>
		<meta charset="utf-8">
		<title>Beispiel für Das Einbinden in die Header-Datei</title>
		<meta name="description" content="">
        <style>
        h1 {
        color: red;
        background-color: black;
        }
        </style>
	</head>
	<body>
	<body>
</html>



    Einbindung der CSS-Datei im Kopf-Bereich
    wirkt sich auf das ganze HTML-Dokument aus

internes Einbinden


    Beispiel Attribut: 
    

    Überschrift

    Widerspricht dem Grundprinzip von HTML, Struktur und Gestaltung zu trennen.

externes Einbinden


    Auslagern in eine externe Datei z.B. meindesign.css

    Der Verweis in der HTML Datei sieht dann so aus:
     
    
<!DOCTYPE html>
<html>
	<head>
	  <meta charset="utf-8">
	  <title>Einbinden einer externen CSS-Datei im Head-Bereich</title>
	  <meta name="description" content="">
      <link rel="stylesheet" type="text/css" href="meindesign.css">
	</head>
	<body>
	<body>
</html>
