Was sind JS-Frameworks

JS Frameworks

  • Deutsch: "Rahmenstruktur"
  • Programmiergerüst, d.h. kein fertiges Programm
  • Codestrukturen zur Erstellung von Anwendungen
  • Bibliotheken
  • Basisbausteine des Frameworks: abstrakte und konkrete Klassen
  • Ziel: kein direktes Arbeiten mit aufwändigem DOM Scripting
  • Zusammenfassung: Frameworks bieten einfache Methoden und Objekte als eine Abstraktionsschicht über JS an

Was sind JS-Frameworks

Ereignisgesteuerte Frameworks

  • Reaktion auf ein bestimmtes Event
  • nicht selbstgesteuert,d.h. ein bestimmtes Event wird duch einen Vorgang ausgelöst

Aktionsgesteuerte Frameworks

  • Reaktion durch Selbststeuerung
  • Aktionen lösen Methodenaufrufe aus
  • Beispiel: Mausklick, Button, Tastaturanschläge

Was sind JS-Frameworks

Vorteile

  • Einfache Bedienung
  • Helferfunktion
  • Fortgeschrittener Programmierstil möglich
    • Beispiel: funktionale und objektorientierte Programmierung
  • In den meisten Fällen kostenlos
  • Code frei verfügbar
  • einfach einzubinden
  • Weniger eigene Schreibarbeit
  • Fertige Widgets und Animationen
  • Code anpassbar

Was sind JS-Frameworks

Nachteile

  • Intensive Beschäftigung mit dem Framework
  • Voraussetzung: Vorkenntnisse in JS
  • Verbreitete Frameworks: mangelnde Dokumentation und Anleitung
  • Frameworks haben unterschiedliche Syntax
  • Inkompatibilitäten bei paralleler Nutzung von JS-Frameworks

Bekannte JS-Frameworks

jQuery

  • meist verbreitete JS-Bibliothek
  • Ständige Weiterentwicklung
  • Viele Plug-ins und Erweiterungen
  • Komfortable DOM-Manipulation und -Navigation
  • Ajax-Funktionalitäten

Bekannte JS-Frameworks

Mootools

  • Hohe Browserkompatibilität
  • Gute Dokumentation
  • starke Objektorientierung
  • Erweiterbar und modular

Bekannte JS-Frameworks

Dojo Toolkit

  • Freie Javascript-Bibliothek
  • Hauptteil Dojo stellt normale Werkzeuge wie jede JS-Bibliothek zur Verfügung.
  • 2 Teil Dijit enthält vorgefertigte Komponenten um grafische Benutzeroberflächen nach dem Baukastensystem zu erstellen
  • 3 Teil DojoX stellt Komponenten zur Verfügung die allerdings teilweise als experimentell anzusehen sind.

Bekannte JS-Frameworks

Yahoo User Interface Library

  • Wurde von Yahoo-Programmierern weiterentwickelt
  • Seit Januar 2009 kann auch die Entwicklergemeinschaft an YUI mit Hilfe von GitHub zur Weiterentwicklung beitragen
  • Im Mittelpunkt steht die Kompatibilität auf allen Browsern
  • Techniken wie AJAX und Dynamic HTML werden benutzt

Bekannte JS-Frameworks

ExtJS

  • Ursprünglich: Sammlung von Funktionserweiterungen für die YUI
  • Mit der Zeit: unabhängigen Bibliothek
  • Clientseitiges JavaScript- bzw. Ajax-Framework
  • Für komplexe Webanwendungen
  • Umfangreiche Sammlung von Steuerelementen
  • Hohe Browserkompatibilität

Bekannte JS-Frameworks

Mobile

  • jQuery Mobile
  • Titanium Mobile
  • The-M-Project
  • Dojo Mobile
  • Sencha Touch

Warum benötigt man JS-Frameworks für Mobile Anwendungen?

  • Unterschiedliche Funktionen
  • Smartphones: Mausklicke und Scrollen wird durch das "Touchen" ersetzt
